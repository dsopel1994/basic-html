HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is used to define our content, structure, and meaning. You don't use it for styling purposes. You use CSS instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content, and closing tag
  • You can also have void(empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but no worries, you'll learn them with experience.

    • You can learn about all available HTML elements on the MDN HTML element reference

CSS Summary

CSS(Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication you typically use global styles instead
  • Alternatively, you can use external style sheets as well which you link to
  • When working with CSS concepts such as inheritance, specificity, and the box model should be understood

    • Learn more about all available CSS properties on the available MDN CSS reference
